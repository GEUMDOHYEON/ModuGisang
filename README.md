# ☀️ 서비스 개요
- 사용자 1인 또는 다수가 **미라클 모닝** 습관을 형성하기 위해, **모두기상** 앱에서 ‘N일 동안 일찍 일어나기 챌린지’를 진행
- 매일 **기상시간**이 도래하면, **잠 깨기에 도움이 되는 스피드 게임**에 참여하며, 챌린지 목표 달성
